Objective: Perform Agglomerative clustering on the fleet driver data dataset with an appropriate number of clusters. Plot the dendrogram and Compare the results from your K-Means model and your Hierarchical clustering model using silhouette scores.

plt.tight_layout()
plt.show()

The above shown plot describes Hierarchical Clustering Dendrograms> This is a Bottom-Up Approach and we select the number of clusters by looking at Height of each clades. The more the height of clades(Vertical) we select that many clusters
